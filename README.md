# ☀️ SolarFlow Premium Marketplace

A high-end solar energy eCommerce platform built with Next.js 14, MongoDB, and Tailwind CSS.

## 🚀 Quick Setup

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)

### Installation

```bash
# Clone and install
git clone <repo>
cd solarflow
npm install

# Configure environment
cp .env.local.example .env.local
# Edit .env.local with your MongoDB URI

# Seed database with sample data
npm run seed

# Start development server
npm run dev
